# Code of Conduct

NeoTreescan adopts the **Contributor Covenant, version 2.1** as its code of
conduct. All contributors, maintainers, and participants in project spaces
(issues, pull requests, discussions, email) are expected to uphold it.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

To report a violation, email
[contact@neortal.com](mailto:contact@neortal.com). Reports are handled
confidentially by the maintainers at Neortal Technologies Inc. Expect an
acknowledgment within 3 business days.

For security issues specifically, see [SECURITY.md](SECURITY.md) instead.

## Enforcement

Maintainers follow the Contributor Covenant's enforcement ladder: correction,
warning, temporary ban, permanent ban. Decisions are final but may be
appealed in writing to the same address.
